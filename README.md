# E-Commerce-Business-Analysis
End-to-end analysis of Brazilian Olist E-Commerce dataset including EDA and business insights.


                                          Project Overview:

This project presents a full-cycle data analysis of the Brazilian Olist E-Commerce dataset.

° Key Analysis Performed °

- Data cleaning & preprocessing
- Monthly sales trend analysis
- Top product categories by revenue
- Customer segmentation (one-time vs repeat buyers)
- Delivery delay impact on review scores

° Key Business Insights °

- Orders show consistent growth over time
- Beauty & Health category generates highest revenue
- Delivery delays significantly reduce review scores
- Customer retention opportunity exists

° Tools Used °

- Python
- Pandas
- Matplotlib
- Seaborn
